### 轮回

- M靠着(公司+职级)背景或者跪舔哀求大佬拿到推荐，忽悠进下一个大冤种
- 如果不是leader岗，M的能力落不了地，没啥产出，就得跪舔哀求leader，不折手段没底线地苟下去，各种表面上“搞关系情商高”，靠近后收集弱点，找准机会背后捅刀，等待leader位置空出来的机会
- leader位置空出来后，M再找+2，拼命包装自己，吹甚至骗，同时跪舔，没尊严，不要脸，让+2觉得自己不管怎么样都会是一条忠心的狗；另一方面打其他人小报告，说其他人都有问题。从而坐上+1的位置
- M当上leader后，通过打小报告，排挤等手段把有能力，有威胁，知道自己名声烂的人搞走，同时接收这些人的工作成果
- 能力好的，性格硬的迟早都会主动或被动恶心走，M带着一群能力不如自己，没啥威胁的下属，安安稳稳地当上老白兔
- 因为吃老本，慢慢没有竞争力，慢慢做不下去，留下一个烂摊子，M再换一个地方，进下一个轮回
